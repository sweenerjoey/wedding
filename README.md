# Joey Sweener and Megan Labosky Wedding Website

Repository for the files associated with [joeyandmegan.com](joeyandmegan.com), our wedding website programmed and designed from scratch by yours truly, and hosted through [GitHub Pages](https://pages.github.com/). The design pairs large, elegant serif typography with a more utilitarian sans-serif, and features various Monet paintings sourced from the National Gallery of Art in Washington D.C. (at least until we have engagement photos!).

Thanks for checking it out.
